# Featured Projects

A fully interactive 3D portfolio world built using `three.js`, `Blender`, and spatial web audio, combining creative development, game-inspired interactions, and immersive storytelling into a playful portfolio experience.

## Inspiration
Inspired by Bella Xu’s Folio and recreated as part of my journey into creative web development with three.js and Blender through this [intro course](https://www.youtube.com/watch?v=X3pPAdQBKHo).

Bella’s original Folio received the Special Kudos and Best UI/UX & Innovation awards from CSSDA! Congratulations!! ❤️❤️❤️
